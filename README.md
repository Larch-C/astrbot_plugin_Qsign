# 二次元签到插件

**基础功能**
 - 签到
 - 签到查询
 - 赎身
 - 购买@用户
 - 出售@用户
 - 存款[空格][金币数量]
 - 取款[空格][金币数量]

# 注意
**应该仅支持qq客户端，且非官方bot**
- 牛牛插件会消耗用户金币
- 除购买以及出售用户外，其他指令需要前缀或者@bot
# 玩法
- 签到获得金币（根据基础身份、连续签到、黑奴数量、银行利息等判定具体可通过签到查询指令看预计奖励）
- 根据金币（身价）获得身份
- 不同身价每天签到基础值不同，不同身价的黑奴提供的加成不同
- 被购买后，身份改为[黑奴]每天签到收益减少50%
